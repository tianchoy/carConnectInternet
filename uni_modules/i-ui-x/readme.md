# I UI X

I UI X 是一个面向 UniAppX 的 `uni_modules` 组件库插件，组件以 `.uvue` 单文件组件形式提供，适合在移动端页面、小程序页面和 H5 页面中通过 easycom 自动引入使用。

本组件库依据 DCloud uni-app x 规范开发，优先使用 UniAppX 内置组件与 API，尽量保持组件结构轻量、参数命名清晰、交互行为真实可用。

## 插件信息

- 插件名称：`i-ui-x`
- 插件类型：前端组件 / 通用组件
- 组件前缀：`i-`
- 组件目录：`src/uni_modules/i-ui-x/components`
- 演示页面：`src/subpkg`
- 适用场景：UniAppX 项目的基础 UI、表单输入、数据展示、反馈交互、布局排版、导航入口和常用扩展能力。

## 安装使用

将本目录放入项目的 `src/uni_modules/i-ui-x` 下，UniAppX 会自动识别 `components/*/*.uvue` 组件。页面中可直接使用组件标签：

```vue
<template>
  <i-button type="primary">确认</i-button>
  <i-alert type="success" title="保存成功" description="操作已完成"></i-alert>
</template>
```

颜色、尺寸、状态、事件等能力请以组件演示页中的「效果演示」「Props 调试」「API」为准。

## 基础组件

- `i-color`：语义色板组件，支持点击、复制、禁用状态，以及主色、浅色、深色、禁用色等颜色展示。
- `i-icon`：图标组件，支持字体图标、图片地址、颜色、大小、旋转、旋转动画和点击事件。
- `i-image`：图片组件，支持裁剪模式、圆形/方形/圆角形状、懒加载、预览、加载中、加载失败和事件回调。
- `i-button`：按钮组件，支持类型、尺寸、形状、朴素、细边框、加载中、图标、颜色、节流点击和开放能力事件透传。
- `i-text`：文本组件，支持内容、字号、语义类型、颜色、装饰线、块级显示、加粗和点击事件。
- `i-row` / `i-col`：栅格布局组件，支持 24 栅格、间距、弹性对齐、换行、偏移、推拉和内边距。
- `i-cell`：单元格组件，支持头像/图标、标题、描述、右侧文字、卡片/普通模式、边框、跳转、插槽和点击事件。
- `i-badge`：角标组件，支持点状、数字、文本、最大值、位置、偏移、隐藏、字号、颜色和点击事件。
- `i-tag`：标签组件，支持语义类型、尺寸、样式、圆角、自定义颜色、图标、渐变、可关闭、禁用和事件。
- `i-loading`：局部加载组件，支持图标旋转、文字、横向/纵向排列、隐藏文字、语义颜色和显示控制。
- `i-loading-page`：加载页组件，支持全屏加载、遮罩、点击遮罩关闭和状态同步。

## 表单组件

- `i-form` / `i-form-item`：表单与表单项组件，支持模型数据、规则校验、必填校验、横向/纵向标签、操作按钮、实时状态、错误滚动和校验方法。
- `i-calendar`：日历组件，支持单选、多选、范围选择、默认日期、最小/最大日期、内联或弹层模式、主题色和确认/关闭事件。
- `i-keyboard`：数字键盘组件，支持弹层显示、长度限制、最大值限制、小数/整数、密码模式、长按和主题色。
- `i-cascader`：级联选择组件，支持树形数据、多级选择、禁用项、内联/弹层、列宽、横向自动滚动、自定义头部和选择事件。
- `i-picker`：选择器组件，支持滚轮选择、单列/多列、默认索引、默认值、加载状态、工具栏、输入触发和确认/取消事件。
- `i-datetime-picker`：日期时间选择器，支持日期、时间、年月、日期时间模式，最小/最大时间限制、时间范围限制和弹层确认。
- `i-color-picker`：颜色选择器，支持 RGB、光谱、色卡三种模式，透明度、HEX 输入、颜色预览、推荐颜色和弹层确认。
- `i-select`：选择器组件，支持字符串/对象选项、弹层位置、可清空、标签、禁用项、边框、文字对齐和值同步。
- `i-rate`：评分组件，支持数量、最低分、半星、只读、禁用、图标、颜色、尺寸、间距和评分事件。
- `i-search`：搜索组件，支持形状、清空、右侧操作、左右插槽、图标点击、搜索、聚焦、失焦和值同步。
- `i-number-box`：步进器组件，支持最小值、最大值、步长、整数/小数、异步变更、按钮禁用、输入禁用和自定义插槽。
- `i-upload`：上传组件，支持图片选择、预览、删除、数量限制、缩略图间距、自定义上传区域、自定义文件区域和读文件事件。
- `i-input`：输入框组件，支持前后插槽、清空、密码显示切换、字数统计、边框、形状和原生输入事件。
- `i-textarea`：多行输入组件，支持字数统计、自动高度、边框、背景、圆角和原生多行输入事件。
- `i-checkbox`：复选框组件，支持布尔值或数组值、方形/圆形/按钮等形态、插槽、标签控制、颜色和选中事件。
- `i-radio`：单选框组件，支持按名称匹配、圆形/方形/点状/按钮等形态、插槽、标签控制、颜色和值同步。
- `i-switch`：开关组件，支持加载中、禁用、形状、自定义开关值、异步变更、文字标签、颜色和事件。
- `i-slider`：滑块组件，支持单值/区间、最小值、最大值、步长、禁用、只读、禁止交叉、颜色、显示数值和拖动事件。
- `i-album`：相册组件，支持多图展示、键名配置、单图/多图尺寸、裁剪模式、最大数量、溢出提示和图片预览。
- `i-slider-button`：滑动按钮组件，支持轨道尺寸、圆角、轨道文字、阈值、自动复位、滑块插槽、成功/重置/变更事件和重置方法。

每个组件演示页通常包含三个页签：

- `效果演示`：展示组件的典型使用场景，要求真实使用组件能力。
- `Props 调试`：覆盖组件真实支持的 props，修改后应立即影响演示效果。
- `API`：展示组件 Props、Events、Slots、Methods，内容来自本地 API 注册表。

新增或调整演示页时，需要同步维护 `src/pages.json` 和首页入口。

## 设计说明

I UI X 的组件 API 追求清晰、直接、可验证。组件参数应与实际能力一致，不保留看起来支持但没有真实功能的 props 或 events。

视觉风格以移动端稳定显示为优先：文本、图标、按钮内容需要垂直居中；卡片、列表、弹层、表单项、表格和导航区域保持统一间距；小程序与 H5 都要避免遮挡、溢出、错位或无法点击。

组件源码应保持 `.uvue` 格式，使用 Vue 3 Composition API 风格组织逻辑。除非明确要求，不引入第三方状态管理或第三方 UI 框架。

## 发布说明

发布到 DCloud 插件市场时，建议只打包 `src/uni_modules/i-ui-x` 插件目录，不包含项目根目录、`pages.json`、`App.uvue`、`main.uts`、`unpackage`、`node_modules` 或版本控制目录。

插件信息以 `package.json` 为准，组件能力说明以本文件、演示页和 API tab 为准。

使用过程中如果有疑问，可评论留言。
