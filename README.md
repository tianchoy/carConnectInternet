- 项目介绍
  - 项目名称：carConnectInternet
  - 项目描述：这是一个基于uni-app-x的车辆定位器项目，用于车辆定位。
  - 项目架构:
    - 前端：uniAppX + Vue3 + TypeScript + uv-ui
    - 后端：Java + Spring Boot
    - 数据库：MySQL

- 项目功能
  - 微信登录 + 账号密码登录
  - 定位器定位车辆位置
  - 车辆定位器状态监控
  - 车辆行驶轨迹回放
  - 车辆行驶里程统计
  - 车辆行驶时间统计
  - 地图电子围栏设置
  - 车辆跟踪功能
  - 物联网卡续费

- 使用方法
  - 下载项目代码
  - 安装依赖
  - 运行项目
  - 登录后添加车辆定位器IMEI号
  - 定位车辆位置
  - 监控车辆定位器状态
  - 回放车辆行驶轨迹
  - 统计车辆行驶里程和时间
  - 设置地图电子围栏
  - 续费物联网卡
